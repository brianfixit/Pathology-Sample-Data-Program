Olympic Sample Data Driven Web Site

by Brian Harris , July 2014

The purpose of this web site is to transform sample data related to the 2012 Olympic games and use it as the basis for creating SVG graphics using D3.js. The results are then to be displayed using Twitter BootStrap and javaScript

All of this site is done with with CSS, JavaScript And D3.js, Not ONE Image is used within!
